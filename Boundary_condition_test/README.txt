Readme file for Boundary condition test:

This folder contains necessary files to run a Boundary condition test. 
To turn on/off the test in .yaml file under material section

Use the following command in your terminal:
Peridigm Boundary_condition_test.yaml

After running the input file, you will see the logged test results under "Output.txt" file.
Note:
Since the Output.txt file opens in append mode, 
it needs to be deleted each and every time for the new run.
